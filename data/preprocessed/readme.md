# Preprocessed Data for CDC-ECG Analysis

This folder contains the preprocessed beat-level CSV files required to reproduce all figures and tables in the *Nature Aging* Brief Communication submission (2026).

**Download link (Zenodo):**  
[→ Direct download of preprocessed data](https://zenodo.org/records/19246123)  

**Files included:**
- `ludb_beats.csv`
- `qtdb_beats.csv`
- `ptb_beats.csv`
- `ptbxl_beats.csv`
- `fantasia_beats.csv`
- `autonomic_aging_beats.csv`
- `code15_beats.csv`
- `code15_exams.csv` (metadata for CODE-15% mortality analysis)

After downloading, put all files directly into this folder (`data/preprocessed/`).  

**File sizes:** <500 MB total 

**Note:** The original raw databases (HDF5/WFDB) are **not** provided here to keep the download small. They are only needed if you want to re-run the export scripts from scratch.
